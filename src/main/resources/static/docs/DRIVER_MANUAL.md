## Managing Packages

### Pickup Process

1. **Navigate to Pickup**:
   - Click **"Navigate"** on package
   - Use GPS navigation
   - Arrive at pickup location

2. **Collect Package**:
   - Verify package details
   - Scan or enter tracking number
   - Take pickup photo (if required)
   - Get sender signature (if required)
   - Click **"Package Collected"**

3. **Package Status**:
   - Automatically updates to "Collected"
   - Sender receives notification
   - Start delivery process

### Delivery Process

1. **Navigate to Delivery**:
   - Follow optimized route
   - Click **"Navigate"** for directions
   - Arrive at delivery address

2. **Deliver Package**:
   - Verify recipient details
   - Confirm package condition
   - Take delivery photo
   - Get recipient signature
   - Click **"Delivered"**

3. **Delivery Completion**:
   - Status updates to "Delivered"
   - Recipient receives notification
   - Earnings updated
   - Rate visible in dashboard

### Failed Delivery

If delivery fails:
